## [v1.0.0] - 2023년 (이전 버전)

### 🏫 중앙 정보처리 학원 머신러닝 프로젝트 (팀 프로젝트)

**프로젝트명**: 인공지능을 활용한 2023 국제 e-sports 대회 출전 한국팀 솔루션

**팀원**: 장휘재, 김기범, 손수한

### 프로젝트 개요
- **목적**: 2023 롤드컵 출전 한국팀 승패 예측 및 전략 제공
- **배경**: 2023년 서울 고척돔에서 열리는 롤드컵 대비
- **데이터**: 2022년 선수별 경기 데이터 (Games Of Legend, op.gg)
- **모델**: Random Forest (앙상블 학습)

### 주요 특징
1. **선수별 분석**
   - 5개 포지션 (Top, Jungle, Mid, ADC, Support) 각각 모델링
   - 개별 선수 데이터 크롤링 및 전처리
   - 팀 데이터와 선수 데이터 분리 분석

2. **데이터 전처리**
   - 웹 크롤링으로 팀별/선수별 데이터 수집
   - 딕셔너리 기반 URL 관리
   - 문자형 → 숫자형 변환
   - CSV 파일로 저장

3. **모델링 방법**
   - RandomForest Classifier
   - GridSearchCV로 하이퍼파라미터 튜닝
   - 교차 검증 (Cross-validation)
   - 앙상블 평균법으로 최종 예측

4. **변수 중요도 분석**
   - **선수별**: K+A Per Minute, GPM, Assists, Damage dealt to turrets/buildings
   - **팀별**: towerslost (0.47), gdm (0.17), gpm (0.09), kd (0.08)

### 결과
**2023년 LCK 팀별 승패 예측 정확도**:
- T1: 89.04%
- Gen.G: 79.73%
- Dplus KIA: 95.45%
- DRX: 97.73%
- KT: 93.33%
- Liiv SB: 92.16%
- NS: 95.24%
- HLE: 92.98%
- KDF: 86.67%
- BRO: 93.19%

### 프로젝트 결론
1. 한국팀뿐 아니라 다른 나라 선수 데이터로 상대팀 전략 예측 가능
2. 상대팀 데이터와 아군 데이터를 함께 활용하여 새로운 전략 수립 가능
3. 선수 개인 컨디션과 순간 판단력의 중요성 확인

### v1.0과 v2.0의 주요 차이점

| 항목 | v1.0 (팀 프로젝트) | v2.0 (개인 프로젝트) |
|------|-------------------|---------------------|
| **목적** | 롤드컵 한국팀 전략 제공 | 실전 배포 가능한 예측 모델 |
| **데이터 범위** | LCK 10개 팀 선수별 | LCK 전체 976경기 |
| **분석 단위** | 선수별 + 팀별 분리 | 경기별 통합 분석 |
| **평가 방법** | 교차 검증 | Time-based split |
| **Data Leakage** | 방지 노력 | 2단계 완전 차단 |
| **정확도** | 89-97% | 69.05%  |
| **결과물** | 발표 자료 (PPT) | GitHub 포트폴리오 |
| **문서화** | 참고문헌 10개 | 전문 문서 6개 + 학술 비교 |
| **재현 가능성** | 중간 | 높음 (전체 코드 공개) |

### v2.0에서 개선된 점
1. **방법론 정확성**: Data Leakage 완전 차단
2. **실전 적용성**: Time-based split으로 실제 시나리오 시뮬레이션
3. **문서화**: 전문적인 포트폴리오 수준
4. **재현성**: 전체 코드 및 프로세스 공개
